Asqav Shadow AI Capture

Configure the API key and synthetic agent ID issued by your Asqav admin. Metadata-only receipts are emitted to api.asqav.com when a managed user navigates to a known AI tool.

Credentials

The API key is stored in chrome.storage.session (in-memory, cleared on browser restart) so it is not readable by other extensions with the storage permission. Re-enter after each browser restart, or rotate via the Asqav dashboard if you suspect compromise.

AI tool detection

Detection of navigation to AI tools requires host permissions for the seed list of AI domains. Click below to grant these at runtime. The extension cannot read or modify pages on any other domain.
